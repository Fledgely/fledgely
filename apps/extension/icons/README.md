# Extension Icons

This directory contains the extension icons for the Fledgely Chrome extension.

## Required Sizes

- `icon16.png` - Toolbar icon (favicon size)
- `icon32.png` - Windows task bar
- `icon48.png` - Extensions management page
- `icon128.png` - Chrome Web Store

## Current Status

**Placeholder icons** - These are simple colored placeholder icons for development.
Replace with branded icons before Chrome Web Store submission.

## Generating Icons

1. Create your icon design at 128x128 or larger
2. Export at all required sizes: 16, 32, 48, 128
3. Ensure icons are square PNG files with transparency support

## Design Guidelines

- Use a simple, recognizable symbol
- Ensure visibility at 16x16 size
- Follow Chrome Web Store icon guidelines
- Consider both light and dark browser themes
